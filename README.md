# ABN TV

Modern single-page application for browsing TV shows, seasons, episodes and cast information.
Built with **Vue 3**, **Vite**, **Pinia**, **Vue Router** and **TypeScript**.



## Prerequisites

- **Node.js** ≥ 20.19.0 or ≥ 22.12.0  
  (We recommend using the latest LTS version)

## Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/srvvsitaramaraju-afk/abn-tv.git
cd abn-tv

# 2. Install dependencies
npm install

# 3. Create local environment file (if not already present)
cp .env.example .env.local

# 4. Run development server
npm run dev

# 5. Run unit tests
npm run test:unit

# 6. Generate coverage report
npm run coverage

#7 . Type issues
npm run type-check
